# 개인 정보 JavaScript 라이브러리 사용

Privacy JavaScript 라이브러리를 사용하여 응용 프로그램에서 고객 ID를 수집한 후 Privacy Service에 보내는 방법을 알아봅니다. 예를 들어 데이터 수집(이전의 Launch)에서 태그 확장을 설정하는 방법을 알아봅니다.
