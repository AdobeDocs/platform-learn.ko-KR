# 세그먼트 일치 수신 데이터

세그먼트 일치 기능을 사용하면 전략적 파트너가 데이터를 공유할 수 있습니다. 이 비디오에서는 데이터를 승인하고 수신하는 방법과 데이터를 보고 고유한 세그먼트에 추가할 수 있는 위치에 대해 알아봅니다. 자세한 내용은 [설명서](https://experienceleague.adobe.com/ko/docs/experience-platform/segmentation/ui/segment-match/overview){target="_blank"}를 참조하세요.

>[!IMPORTANT]
>
>Adobe은 고객이 공동 작업을 수행하고 대상을 교환할 수 있도록 2021년에 세그먼트 일치 를 도입했습니다. 2025년 초, Adobe은 이 사용 사례를 충족하기 위한 보다 장기적인 접근 방식인 Real-Time CDP Collaboration을 도입했습니다.
>
> * 미국, 캐나다, EMEA, 오스트레일리아 및 뉴질랜드의 고객을 위해: Adobe은 Real-Time CDP Prime 및 Ultimate 고객에게 세그먼트 일치에서 Real-Time CDP Collaboration으로 데이터 공동 작업 사용 사례를 전환할 것을 권장합니다. Real-Time CDP Collaboration에 대한 [설명서](https://experienceleague.adobe.com/ko/docs/real-time-cdp-collaboration/using/home){target="_blank"} 및 [빠른 시작 안내서](https://experienceleague.adobe.com/ko/docs/real-time-cdp-collaboration/using/quick-start-guide){target="_blank"}를 보고 자세한 내용을 알아보려면 Adobe 계정 팀에 문의하십시오.
> * 다른 모든 지역의 고객을 위한: Real-Time CDP Collaboration이 2026년에 해당 지역에 릴리스될 때까지 세그먼트 일치가 권장되는 옵션입니다.
