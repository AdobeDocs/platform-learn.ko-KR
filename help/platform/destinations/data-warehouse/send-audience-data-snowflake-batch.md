# Snowflake 데이터 공유로 대상 데이터 보내기(일괄 처리)

Adobe Real-Time CDP에서 Snowflake 배치 대상을 구성하고 사용하여 매일 전체 새로 고침 대상 스냅샷을 Snowflake 계정에 제공하는 방법에 대해 알아봅니다. Adobe Experience Platform의 통합 워크플로우, Snowflake에서 데이터가 동적 테이블로 표시되는 방법, 성공적인 통합을 위한 주요 설정 단계 및 유효성 검사를 참조하십시오.

자세한 내용은 [설명서](https://experienceleague.adobe.com/ko/docs/experience-platform/destinations/catalog/warehouse/snowflake-batch)를 검토하세요.
