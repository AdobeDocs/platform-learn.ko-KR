# 쿼리 실행

이 비디오에서는 Adobe Experience Platform 인터페이스 및 PSQL 클라이언트에서 쿼리를 실행하는 방법을 보여줍니다. 또한 XDM 개체에서 개별 속성을 사용하고 Adobe 정의 함수를 사용하고 CREATE TABLE AS SELECT(CTAS)를 사용하는 것이 나와 있습니다. 자세한 제품 설명서는 [쿼리 편집기 UI 안내서](https://experienceleague.adobe.com/docs/experience-platform/query/ui/user-guide.html).
