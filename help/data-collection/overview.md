# 데이터 수집 튜토리얼

데이터 수집은 Adobe의 차세대 웹 사이트 태그 및 모바일 SDK 관리 기능입니다. 데이터 수집은 관련 고객 환경을 향상시키는 데 필요한 분석, 마케팅 및 광고 솔루션을 배포하고 관리하는 간단한 방법을 고객에게 제공합니다. 이 비디오와 튜토리얼을 사용하여 이 중요한 기술을 학습하십시오.
