# 이벤트 전달 확장 추가

데이터 수집에서 이벤트 전달 확장을 추가하는 방법을 알아봅니다. 자세한 내용은 [이벤트 전달 설명서](https://experienceleague.adobe.com/docs/experience-platform/tags/event-forwarding/overview.html)를 참조하세요.

>[!NOTE]
>
>Adobe Experience Platform Launch은 데이터 수집 기술군으로 Adobe Experience Platform에 통합되고 있습니다. 이 콘텐츠를 사용하는 동안 알아야 하는 몇 가지 용어 변경 사항이 인터페이스에 롤아웃되었습니다.
>
> * 이제 platform launch(Client Side)가 **[!DNL tags]**&#x200B;입니다.
> * 이제 platform launch 서버측이 **[!DNL event forwarding]**&#x200B;입니다.
> * 이제 Edge 구성이 **[!DNL datastreams]**&#x200B;입니다.
