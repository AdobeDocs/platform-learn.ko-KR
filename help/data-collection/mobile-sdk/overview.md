# Adobe Experience Platform Mobile SDK 개요

Adobe Experience Platform Mobile SDK가 모바일 애플리케이션에서 전체적인 구현을 수행하는 방법에 대해 알아봅니다. Platform Mobile SDK는 획득하고, 분석하고, 참여하고 최적화하는 데 도움이 됩니다. 성능 기준에 더욱 부합하고, 모듈식이며, 확장 가능하고 데이터 수집과 통합됩니다. 데이터 수집의 정교한 규칙 엔진, 권한 및 게시 워크플로우를 사용하면 Adobe Analytics, Campaign, Audience Manager, Target 및 모바일 앱의 서드파티 솔루션을 효율적으로 구현할 수 있습니다.
