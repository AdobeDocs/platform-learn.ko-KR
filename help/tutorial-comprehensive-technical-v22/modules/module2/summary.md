# 요약 및 이점

Adobe Experience Platform에 대해 배우는 데 시간을 투자하신 것을 축하드립니다.
이 단원에서는 스키마, mixin 및 데이터 세트를 만드는 방법을 배웁니다. Launch를 사용하여 데이터를 스트리밍으로 수집하는 방법과 CSV 파일을 Adobe Experience Platform으로 가져오고 XDM 스키마에 대해 CSV 열을 매핑하여 데이터를 수집하는 방법을 알아보았습니다.

## 이점

Adobe Experience Platform 의 데이터 수집 기능 의 이점을 강조하겠습니다.

- XDM을 단일 언어로 사용하여 경험을 설명하면 데이터 모델링이 수집 시 한 번 발생하고, 여기에서 Adobe Experience Platform과 통합된 모든 애플리케이션에서 데이터 모델링을 쉽게 다시 사용하고 사용할 수 있습니다.
- XDM에는 가능한 한 많이 재사용해야 하는 mixin 형태로 많은 기본 정의가 포함되어 있습니다. 또한 XDM은 특정 브랜드 및 데이터 요구 사항을 충족하도록 XDM을 사용자 지정한다는 목표를 가지고 브랜드가 고유한 혼합을 만들 수 있도록 충분히 유연합니다.
- Adobe Experience Platform에서 만든 데이터 세트는 실시간 고객 프로필을 실시간으로 하이드레이션하도록 구성할 수 있습니다.
- 워크플로우를 사용하여 CSV, JSON 또는 Parquet 파일 형식의 데이터를 Adobe Experience Platform에 쉽게 수집할 수 있습니다
- 데이터 세트 수준의 데이터 거버넌스 컨트롤을 사용하여 Adobe Experience Platform에서 한 번만 데이터 레이블을 지정할 수 있습니다. 그러면 해당 레이블은 Adobe Experience Platform에서 데이터를 사용하는 방법과 Adobe Experience Platform에 연결된 다른 애플리케이션에서 데이터를 사용하는 방법에 영향을 줍니다.
