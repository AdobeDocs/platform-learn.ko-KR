# 1.7 Adobe Commerce용 지능형 개발자 도구

## 학습 목표

이 모듈에서는 Cursor.ai와 같은 지능형 개발자 도구를 사용하여 Adobe Commerce as a Cloud Service 환경에 대한 확장을 개발합니다. 해당 확장의 목표는 들어오는 주문 이벤트를 서드파티 엔드포인트로 전달하는 것입니다. Adobe Commerce as a Cloud Service의 이벤트 전달은 Adobe I/O App Builder, Adobe I/O Events 및 Adobe I/O Runtime을 사용합니다. 이러한 모든 서비스의 구성은 Cursor.ai의 도움을 받습니다.

## 전제 조건

- Adobe Commerce as a Cloud Service 액세스
- Adobe Experience Manager as a Cloud Service 액세스.
- 개발자 또는 시스템 관리자 액세스를 통해 Adobe I/O에 액세스합니다.
- Cursor.ai에 액세스
- 종속성이 있으므로 이 모듈을 완료했어야 합니다. [Adobe Experience Manager Cloud Service 및 Edge Delivery Services](./../../../modules/asset-mgmt/module2.1/aemcs.md).
- 종속성이 있으므로 이 모듈을 완료했어야 합니다. [Adobe Commerce as a Cloud Service](./../../../modules/asset-mgmt/module1.5/accs.md).
