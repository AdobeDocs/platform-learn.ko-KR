# Analytics 및 에이전트

## 학습 목표

이 모듈에서는 adobe가 아닌 외부 에이전트를 사용하여 Adobe Analytics 및/또는 Customer Journey Analytics에 있는 데이터를 분석합니다.

## 전제 조건

- AA(Adobe Analytics) 및/또는 Adobe Customer Journey Analytics(CJA)에 액세스
- 유료 구독으로 ChatGPT 및/또는 Claude.ai에 액세스

## 연습

[1.5.1 CJA 및 ChatGPT(MCP 서버 포함)](./ex1.md){target="_blank"}

이 연습에서는 ChatGPT에서 사용자 지정 MCP 서버를 설정하여 ChatGPT를 Customer Journey Analytics에 직접 연결하고 ChatGPT에서 직접 Adobe Customer Journey Analytics(CJA)에 호스팅되는 데이터를 분석합니다.

[1.5.2 MCP 서버가 있는 CJA 및 Claude.ai](./ex2.md){target="_blank"}

이 연습에서는 Claude.ai에 맞춤형 MCP 서버를 설정하여 Claude.ai를 Customer Journey Analytics에 직접 연결하고 Adobe Customer Journey Analytics(CJA)에서 호스팅되는 데이터를 Claude.ai에서 직접 분석합니다.

[1.5.3 Adobe Analytics 및 ChatGPT(MCP 서버 포함)](./ex3.md){target="_blank"}

이 연습에서는 ChatGPT에 사용자 지정 MCP 서버를 설정하여 ChatGPT를 Adobe Analytics에 직접 연결하고 ChatGPT에서 직접 Adobe Analytics(AA)에 호스팅된 데이터를 분석합니다.
