# 시작하기 - GenStudio

이 모듈에서는 환경과 컴퓨터가 GenStudio Tech Labs를 시작할 준비가 되도록 모든 것을 설정합니다.

## 학습 목표

- Chrome 확장을 설정하는 방법 알아보기
- Adobe I/O 프로젝트 및 Postman 또는 포스트버스터 설정
- GenStudio Tech Labs 촬영에 필요한 애플리케이션 설치

## 연습

[Experience League 설명서용 Chrome 확장 설치](./ex1.md)

이 연습에서는 자습서 콘텐츠를 개인화하는 데 사용할 수 있도록 Chrome 확장을 설치하고 구성합니다.

[Adobe I/O 프로젝트 구성](./ex2.md)

이 연습에서는 다양한 Adobe API 엔드포인트와 상호 작용하는 데 사용할 Adobe I/O 프로젝트를 구성합니다.

[옵션 1: Postman 설정](./ex3.md)
